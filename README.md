# 📝 RegistrationForm

Projeto **RegistrationForm** desenvolvido para criar uma interface de formulário de cadastro moderna utilizando apenas **HTML e CSS**. O foco do projeto é trabalhar a estrutura de formulários e estilização de inputs.

## 📌 Sobre o Projeto

O **RegistrationForm** é um formulário de cadastro que permite ao usuário inserir informações como nome, email, senha e outros dados. O layout foi desenvolvido com design limpo, centralizado e organizado.

Este projeto foi criado com o objetivo de praticar:

* Estruturação de formulários
* Estilização com CSS
* Organização de inputs
* Layout moderno
* Boas práticas de HTML

## 🚀 Tecnologias Utilizadas

* HTML5
* CSS3
